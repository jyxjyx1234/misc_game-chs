# 使用方法

将patch.xp3和data.xp3放入游戏目录，从原程序启动即可。无需转区。

如果遇到启动问题：

1. 本补丁不支持模拟器运行
2. 尝试：https://aka.ms/vs/17/release/vc_redist.x86.exe 下载安装完后重启
3. 尝试删除存档文件夹后重新启动

重要提示：本作的skip是空格+ctrl（感谢 Steins;Gate）
