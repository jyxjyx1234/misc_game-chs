## 使用说明：

将本补丁中所有文件复制到游戏本体安装目录即可。

基于1.02版进行制作，请自行安装好1.02版补丁。

安装提供的字体可以得到更好的游戏体验（不确定这个字体其他电脑上有没有所以附带上了）。

配置了自动转区，无需手动转区。
