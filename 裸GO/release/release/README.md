# 使用方法

本补丁只支持 Let's 裸 GO！ ぼくラの自由研究 OS変更廉価版（2019年发售）！

将LRG_chs.exe放入游戏目录即可，从LRG_chs.exe启动游戏，无需转区。

如果游戏字体异常，请自行寻找Microsoft YaHei UI Bold字体。

【娱乐用】 做了一个写着玩的小功能，游戏运行后会在游戏目录下生成一个runtime.txt文件，记录游戏的累计用时。
