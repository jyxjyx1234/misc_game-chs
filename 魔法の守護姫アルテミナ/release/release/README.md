## 使用说明：

将本补丁中所有文件复制到游戏本体安装目录即可。**无需转区运行，提供的字体不要安装！**

使用了两个不同版本的免DVD补丁进行打包，如果一个不行可以试试另一个。

有两个版本：简中和繁中（tra），自行选择。

本补丁基于光盘版制作，没有在dl版进行测试。欢迎留言反馈dl版测试的结果。
