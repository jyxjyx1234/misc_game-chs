# 使用方法

使用PIANO_chs.exe启动游戏即可。无需转区。

支持免DVD，但是不支持免安装。

选项在选中后会显示原文本，以便于和攻略对照。制作补丁时，选项文本由ocr自动识别获取，所以选项译文可能会有一些小错误，工作量比较大就没有逐一校对了，发现选项译文有离谱错误可以评论区反馈。
