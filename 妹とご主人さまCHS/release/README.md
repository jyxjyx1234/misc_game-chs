## 使用说明：

本补丁基于Booth上的0.48版进行制作，不适配其他版本。

将本补丁的内容放在游戏目录下，运行 `妹とご主人さま048chspatch_v2.0.exe`安装补丁。运行 `Game.exe`即可启动游戏（无需转区）。

运行uninstall_chs.cmd可以恢复原版。
