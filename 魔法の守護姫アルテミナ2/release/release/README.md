## 使用说明：

将本补丁中所有文件复制到游戏本体安装目录即可。**使用该补丁前如果游戏目录中存在_FONTSET.MED需要删除！****无需转区运行！**

有两个版本：简中和繁中（tra），自行选择。

本补丁基于dl版制作，没有在光盘版进行测试。欢迎留言反馈dl版测试的结果。

如果生肉能够运行，但使用汉化补丁后不能运行，请将生肉的exe提供给我。
