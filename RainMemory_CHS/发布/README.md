## 使用说明：

将本补丁中所有文件复制到游戏本体安装目录即可。会覆盖`script.xp3`这个文件，建议先备份。

安装提供的字体可以得到更好的游戏体验（不确定这个字体其他电脑上有没有所以附带上了）。

配置了自动转区，无需手动转区。
