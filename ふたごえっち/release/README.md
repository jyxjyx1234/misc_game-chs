# 使用方法

在游戏目录下运行 ふたごえっち_chs.exe 以安装补丁，然后从ADV.exe启动游戏。无需转区。

本补丁只支持win7使用，在win7虚拟机中测试成功。

如果启动不成功请先尝试安装 https://aka.ms/vs/17/release/vc_redist.x86.exe 后重启

win10上可以尝试站内的破解补丁，个人测试不成功，暂时没能联系上破解补丁的作者。
